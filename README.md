# Simple Python To-Do List

This project is a beginner-friendly command-line to-do list application. It helps you organize daily tasks by letting you add new tasks, view the current list, delete tasks you no longer need, mark tasks as done, and save everything to a file so your tasks remain available later.

## What the project does

The app provides a simple menu where you can:

- add a new task
- view all existing tasks
- delete a task
- mark a task as completed
- save tasks to a text file

## How to run it

From the project folder, run:

```bash
python main.py
```

## Project structure

- `main.py` - the interactive menu for the app
- `src/simple_project/todo.py` - the to-do list logic and file handling
- `tests/test_todo.py` - simple unit tests for the app

## Example

When you run the program, you will see a menu with options such as adding or viewing tasks. The program stores your tasks in a file named `tasks.txt` in the project folder.
