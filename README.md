# Simple Python To-Do List

This project is a beginner-friendly to-do list app that lets you add, view, delete, and save tasks.

## Run it

```bash
python main.py
```

## Structure

- `main.py` - interactive menu for the app
- `src/simple_project/todo.py` - to-do list logic and file saving
- `tests/test_todo.py` - simple unit tests
